# Developer Box

面向开发者的桌面工具箱

## 使用方法

从 [Release](https://github.com/fhyuncai/developer-box/releases/latest) 下载最新版本即可开始使用

## 开发环境要求

- Node.js 20+
- pnpm 10+

## 本地开发

```bash
pnpm install
pnpm dev
```

## 构建发布文件

```bash
pnpm build
```

输出目录为 `release/`

## 数据路径

- macOS: `~/.developer-box`
- Windows: `C:/Users/<用户名>/.developer-box`
